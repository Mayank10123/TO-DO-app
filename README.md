# 🐾 Tabbie

### Your cute productivity companion for tasks, focus, notes, and daily momentum ✨

Tabbie is a playful productivity dashboard built for people who want structure without a boring workflow. Instead of feeling like a cold admin panel, Tabbie feels like a small desktop companion that helps you organize your day, stay focused, and keep moving. 💡

Inside the app, you can manage tasks, track completed work, run Pomodoro sessions, keep notes, explore activity stats, and move through a clean multi-page workspace designed to feel lightweight and motivating. It also includes onboarding, theme preferences, schedule support, and a dedicated Tabbie experience that gives the project more personality than a standard todo app. 🚀

### 🌟 Highlights

- ✅ Task management with active and completed workflows
- ⏱️ Pomodoro focus sessions for deep work
- 🗓️ Schedule and event views for planning ahead
- 📝 Notes for capturing ideas quickly
- 📊 Activity tracking for tasks and Pomodoro history
- 🎨 Clean and retro themes with light/dark options
- 👋 First-run onboarding for a smoother setup

### 🛠️ Built With

Tabbie is built with React 19, TypeScript, Vite, Tailwind CSS, Radix UI, TipTap, and modern component patterns. The result is a fast, responsive interface with a flexible foundation for future features. It is also easy to extend for personal experiments, polished dashboards, or custom workflow ideas. ⚡

### 🚦 Getting Started

```bash
npm install
npm run dev
```

Then open the local Vite URL in your browser and start setting up your workspace.

### 💭 Why Tabbie?

Because productivity tools should feel helpful, not heavy. Tabbie mixes structure, visual clarity, and a little charm so planning your day feels more human. If you want a project dashboard with personality, this is a strong place to start. 🌈
